# covid19-mortality-data-mining
## Overview
This project analyzes U.S. COVID-19 mortality data to identify trends in deaths by age group,
state, and contributing health conditions. The goal is to support data-driven public health
insights and resource allocation.

## Dataset
- Source: Kaggle (CDC Provisional COVID-19 Deaths Data)
- Size: ~621,000 rows, 13 variables
- Time Period: 2020–2023

## Methods
- Data cleaning and preprocessing using dplyr and tidyr
- Exploratory Data Analysis (EDA)
- Aggregation and comparison across demographics and condition groups
- Visualization using ggplot2

## Tools
- R
- dplyr, tidyr
- ggplot2

## Files
- `/code` – R scripts used for analysis
- `/report` – Final project report (PDF)
- `/data` – Raw dataset
